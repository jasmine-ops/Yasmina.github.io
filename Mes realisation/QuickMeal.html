Quick Meal

À propos de mon site

Lors de mon année de M1, j'ai participé au développement de Quick Meal, une plateforme en ligne permettant aux utilisateurs de consulter les menus de plusieurs restaurants et de passer des commandes. Le back-end a été construit avec PHP Laravel pour gérer les commandes et la logique métier, tandis que le front-end a été conçu avec Bootstrap, HTML et CSS pour offrir une interface moderne et réactive

Technologies utilisées

  • PHP Laravel
  • CSS
  • Bootstrap
  • HTML
